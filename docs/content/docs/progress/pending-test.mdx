# 待测试

- 配置与用户偏好：每个模型渠道新增 OpenAI / Gemini 调用格式选择，默认 OpenAI，切换默认格式时同步更新默认 Base URL；渠道 Tab 增加紧凑醒目的提醒，说明模型是否可选需要到“模型”Tab 选择。
- Gemini 调用格式：支持拉取 Gemini 模型列表，并用于文本问答、在线 Agent 工具调用、基础生图和图生图请求。
- 音频和视频生成：选择 Gemini 调用格式时先给出不支持提示，仍需使用 OpenAI 兼容渠道。
- Logto 登录门禁：未登录访问 `/`、`/canvas`、`/image`、`/video`、`/prompts`、`/assets` 会跳转 `/login?redirect=...`，登录成功后返回原页面，刷新后仍保持登录。
- New API 轻代理：登录后 `/api/new-api/config` 能读取 New API 模型和令牌，配置弹窗打开时自动导入第一个可用令牌为本地渠道，并可手动切换令牌。
- 设置弹窗：New API 区块的“前往 New API 配置”会打开 `/keys`， “关于”按钮会展示原项目、本项目仓库、许可证和版本信息。
- 回归确认：手动多渠道配置、URL 参数导入、拉取模型、WebDAV 同步、提示词和画布功能不应回退。
